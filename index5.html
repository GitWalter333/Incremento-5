Sistema de bombas de agua

Se llevará a cabo un proceso completo de análisis, diseño e implementación de un circuito digital combinacional encargado de controlar dos bombas de agua (B1 y B2) mediante sensores de nivel y temperatura. El objetivo principal será construir un sistema lógico capaz de activar o desactivar las bombas dependiendo de las condiciones establecidas en el problema. Inicialmente, se identificarán las variables de entrada y salida del sistema. Como entradas se utilizarán los sensores de nivel de agua “c” y “d”, además de los sensores de temperatura “T1” y “T2”. Las salidas serán las bombas B1 y B2. Posteriormente, se analizará cuidadosamente el funcionamiento solicitado para comprender en qué situaciones debe activarse cada bomba y cuándo deben apagarse. Después del análisis, se procederá a construir la tabla de verdad del sistema. En esta tabla se representarán todas las combinaciones posibles de los sensores y el comportamiento correspondiente de las bombas. Esto permitirá visualizar de forma lógica cada caso planteado, por ejemplo: cuando el agua esté por debajo del sensor “c”, cuando el nivel esté entre “c” y “d”, cuando el tanque esté lleno o cuando exista fallo en los sensores. También se tomarán en cuenta las condiciones de temperatura que provocan el apagado de cada bomba. Una vez terminada la tabla de verdad, se obtendrán las funciones booleanas para las salidas B1 y B2. Estas funciones representarán matemáticamente el comportamiento lógico del circuito. Luego, se simplificarán utilizando Álgebra de Boole, con el objetivo de reducir la cantidad de compuertas necesarias y hacer el circuito más eficiente. Con las funciones simplificadas, se diseñará el diagrama lógico del circuito utilizando compuertas AND, OR y NOT. En esta etapa se representará gráficamente cómo se conectarán los sensores y las compuertas para producir las señales de salida correctas hacia las bombas B1 y B2. Posteriormente, se realizará la simulación del circuito en un software de diseño digital para verificar que el sistema funcione correctamente en todos los casos posibles. Después de comprobar el funcionamiento lógico, se implementará físicamente el circuito utilizando compuertas lógicas reales, cables de conexión, protoboard y componentes electrónicos necesarios. Finalmente, se documentará todo el proceso en un informe formal con normas APA, incluyendo introducción, objetivos, marco teórico, planteamiento del problema, desarrollo metodológico, tabla de verdad, simplificación de funciones booleanas, diagramas lógicos, implementación física, resultados obtenidos y conclusiones del proyecto.
