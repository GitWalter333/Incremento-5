Bases de Datos

Para el sexto proyecto trabajamos con bases de datos utilizando dos aplicaciones principales: XAMPP y MySQL Workbench. Inicialmente, utilizamos Workbench para diseñar el modelo de una base de datos, creando tablas y estableciendo relaciones entre ellas. Después de finalizar el diseño, generamos el script SQL correspondiente. Posteriormente, mediante XAMPP accedimos desde el navegador a phpMyAdmin, donde pegamos y ejecutamos el script generado previamente para crear la base de datos dentro del servidor local. Finalmente, ingresamos datos en las tablas relacionadas para comprobar que la base de datos funcionara correctamente y que las relaciones entre tablas estuvieran bien estructuradas.
