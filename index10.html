Casa de 3 Niveles

Este proyecto consistió en realizar una simulación de redes utilizando Cisco Packet Tracer. Básicamente, aplicamos los conocimientos aprendidos en el proyecto anterior, pero esta vez dentro de un entorno virtual de simulación. El objetivo era diseñar la red de una casa de tres niveles, conectando múltiples computadoras y dispositivos de red. Cada piso debía trabajar con segmentos IP diferentes para organizar correctamente la comunicación entre dispositivos. Después de realizar todas las conexiones y configuraciones correspondientes, verificamos el funcionamiento de la red enviando paquetes de datos entre las computadoras simuladas dentro del programa. Este proyecto ayudó a comprender mejor el diseño lógico de redes y la administración de direcciones IP.
